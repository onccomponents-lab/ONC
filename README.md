# Website ONC

## Chạy local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Đưa lên GitHub

```bash
git init
git add .
git commit -m "Initial ONC website"
git branch -M main
git remote add origin https://github.com/TEN-TAI-KHOAN/TEN-REPO.git
git push -u origin main
```

## Deploy nhanh

Kết nối repo này với Vercel hoặc Netlify để deploy trực tiếp.
